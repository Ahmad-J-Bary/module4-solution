JavaScript Assignment: Hello & Goodbye Messages

Loop over an array of names and print either 'Hello' or 'Goodbye' to the console based on the first letter of each name

Assignment Instructions

Create a JavaScript program that:

  1. Contains an array of names
  2. Loops through each name in the array
  3. Checks if the name starts with 'j' or 'J'
  4. Prints "Goodbye JSomeName" for names starting with J
  5. Prints "Hello SomeName" for all other names

Implementation Details

The solution uses IIFE (Immediately Invoked Function Expression) to avoid polluting the global namespace.

Key Concepts

Demonstrates array looping, conditional statements, string manipulation, and console output.

JavaScript Code

// Wrap code in an IIFE
(function() {
// Array of names
var names = ["John", "Jane", "Jim", "Alex", "Jasmine", "Mike", "Jessica"];
// Loop through each name
for (var i = 0; i < names.length; i++) {
// Get first letter of the name
var firstLetter = names[i].charAt(0).toLowerCase();
// Check if name starts with 'j'
if (firstLetter === 'j') {
console.log("Goodbye " + names[i]);
} else {
console.log("Hello " + names[i]);
}
}
})();
